## スライド作成時のメッセージ

- DoMINoを「魔法のCFD代替」ではなく、複雑形状上の物理量を学ぶニューラルオペレータとして紹介する。
- 公式フルスケールの再現ではなく、PhysicsNeMoの実クラスを通して構造を観察する縮小教材であると明示する。
- Relative L2約0.51だけを良否判定に使わず、相関、3次元分布、誤差位置を合わせて読む。
- 3分と120分の差は、検証設計の重要性を考える材料として扱い、過学習を断定しない。

## 図版のデザイン要件

- 本文は日本語、Colab生成画像内は文字化け回避のため英語・ASCII表記。
- 3次元車体図を主役にし、表や数式は補助に留める。
- 圧力のReferenceとPredictionは同じカラースケールで比較する。
- 緑をquery点、橙を近傍点として全スライドで統一する。
- 出典は各スライド下部に小さく表示し、公式資料・論文・本リポジトリを区別する。

## スライド着手前に残す選択

内容と素材は揃っている。実際のスライド生成前に決めるのは次の4点だけでよい。

1. 研究所指定のPowerPointテンプレートやロゴの有無
2. 発表時間（20分、30分、45分など）
3. 公式DoMINo構成図を引用するか、自作の簡略図に統一するか
4. GitHub URLをQRコードにするか、短縮せず文字列で示すか
